# Poslatr

Self-hosted social media manager. Composes, stores, schedules, and publishes posts across social platforms. v0.1 builds the spine and proves it end to end with one platform, Mastodon.

## Governance

- Specification: [docs/poslatr-v0.1-prd.md](docs/poslatr-v0.1-prd.md)
- Security policy (binding): [SECURITY.md](SECURITY.md)
- Engineering, security, and review protocol (binding): [CONTRIBUTING.md](CONTRIBUTING.md)

Work proceeds one issue at a time in numeric order (ISS-001 through ISS-012), each behind a PR with CI and a cold principal-engineer review. The tooling in this repo as of ISS-001 is bootstrap-only; the real pnpm + Turborepo workspace lands in ISS-002.
